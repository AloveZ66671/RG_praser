# 形式语言与自动机大作业

本项目实现了正则表达式解析的全流程，包括从正则表达式到最小化 DFA 和正则文法的转换。

---

## 功能简介

本程序支持以下功能：
- 正则表达式解析：RE -> ε-NFA -> NFA -> DFA（含陷阱态） -> 最小化 DFA -> RG（正则文法）。
- 支持的输入符号：
  - **字符**：`0`, `1`
  - **括号**：`()`
  - **运算符**：`+`, `*`
- 输出：
  - 最小化 DFA
  - 正则文法

---
